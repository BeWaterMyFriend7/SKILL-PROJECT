# 软件管理平台 - 原型设计

## 概述

本原型基于《软件管理平台需求分析文档》设计，实现了一个简约大气的软件管理平台界面。

## 设计原则

### 1. 简约大气
- 整体配色协调，以天蓝色为主色
- 页面布局简洁，核心内容居中展示
- 留白适度，避免过度拥挤

### 2. 按钮级别明确
| 按钮类型 | 样式 | 使用场景 |
|---------|------|---------|
| **主要按钮** | 蓝色填充 | 核心操作：下载、搜索、提交 |
| **次要按钮** | 白色带边框 | 辅助操作：详情、返回 |
| **警告按钮** | 黄色填充 | 跳转外部下载 |
| **危险按钮** | 红色填充 | 删除、禁用等危险操作 |
| **成功按钮** | 绿色填充 | 添加用户、启用等 |

### 3. 提示语级别
| 提示类型 | 样式 | 场景 |
|---------|------|------|
| **信息提示** | 蓝色背景 | 一般信息展示 |
| **成功提示** | 绿色背景 | 操作成功反馈 |
| **警告提示** | 黄色背景 | 需要关注的提醒 |
| **错误提示** | 红色背景 | 错误或危险警告 |

## 页面结构

### 1. 软件列表页面
- **搜索栏**：支持关键词搜索、类型筛选、状态筛选
- **视图切换**：卡片视图 / 列表视图
- **软件卡片**：展示软件名称、类型、版本、分类标签
- **快捷操作**：下载 / 跳转下载 / 详情

### 2. 软件详情页面
- **基础信息区**：软件名称、类型、功能简介等
- **版本管理区**：版本列表及下载链接
- **使用统计区**：下载次数、使用人数、趋势图表
- **文档资源区**：使用手册、API文档等
- **用户评价区**：用户评分和评论

### 3. 统计分析页面
- **关键指标**：软件总数、活跃用户、下载量
- **趋势图表**：使用趋势、热门排行、部门分布

### 4. 用户管理页面
- **用户列表**：展示用户信息、角色、部门
- **角色区分**：超级管理员、部门管理员、普通用户
- **权限控制**：编辑、删除、审核操作

### 5. 系统设置页面
- 基本设置、通知设置、权限设置

## 技术实现

- 纯 HTML + CSS + JavaScript
- 响应式布局，支持 16:9 比例
- 无外部依赖，直接浏览器打开即可

## 颜色规范

```css
--primary: #4A90D9        /* 主色 - 天蓝色 */
--primary-dark: #3A7BC8  /* 主色深色 */
--primary-light: #E8F2FC /* 主色浅色 */
--gray-light: #F5F7FA     /* 背景灰 */
--gray: #6B7A8C          /* 次要文字 */
--gray-dark: #3D4852     /* 主要文字 */
--danger: #E74C3C        /* 强调/警告红 */
--success: #27AE60       /* 成功绿 */
--warning: #F39C12       /* 警告黄 */
```

## 字体规范

- 字体：系统默认字体 (-apple-system, BlinkMacSystemFont, Segoe UI, Roboto)
- 标题层级清晰，大小区分明确
- 字体颜色统一，层次分明
