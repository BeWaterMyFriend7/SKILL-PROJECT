# 安全检测流程管理 - 原型设计

## 概述

本原型基于安全检测流程需求设计，实现了一个简约大气的介质安全检测管理界面。

## 设计原则

### 1. 简约大气
- 整体配色协调，以天蓝色为主色
- 页面布局简洁，核心内容居中展示
- 留白适度，避免过度拥挤

### 2. 按钮级别明确
| 按钮类型 | 样式 | 使用场景 |
|---------|------|---------|
| **主要按钮** | 蓝色填充 | 发起检测、确认下一步、完成提交 |
| **次要按钮** | 白色带边框 | 编辑、返回、取消 |
| **警告按钮** | 黄色填充 | 全屏查看 |
| **成功按钮** | 绿色填充 | 完成操作 |
| **危险按钮** | 红色填充 | 删除介质 |

### 3. 提示语级别
| 提示类型 | 样式 | 场景 |
|---------|------|------|
| **信息提示** | 蓝色背景 | 一般信息、操作引导 |
| **警告提示** | 黄色背景 | 需补充信息提醒 |
| **成功提示** | 绿色背景 | 检测通过、完成 |
| **错误提示** | 红色背景 | 检测驳回、错误 |

## 流程设计

### 流程步骤
1. **介质上传** → 上传介质文件，填写字段1、2、3
2. **发起检测** → 点击发起，触发ASG安全检测
3. **结果查看** → 查看通过/驳回状态
4. **使用范围** → 补充使用范围，上传Excel

## 页面结构

### 1. 介质上传页面
- 介质类型选择（安装文件/SDK/镜像）
- 字段1、2、3输入
- 文件上传区域（支持拖拽）
- 已上传介质列表（支持编辑/删除）
- 发起检测按钮

### 2. 检测流程页面
- 任务信息展示
- ASG系统iframe嵌入（预览模式）
- 全屏查看按钮（查看完整内容）
- 提交申请功能
- 刷新状态功能

### 3. 结果查看页面
- 检测结果展示（通过/驳回）
- 检测详情（任务编号、结论、时间、检测项）
- 介质列表及各自状态
- 进入ASG系统查看
- 确认下一步按钮

### 4. 使用范围页面
- 使用范围说明输入
- Excel上传区域（含模板格式说明）
- 已上传Excel展示
- 完成按钮

- 响应式布局
- 无外部依赖，直接浏览器打开即可

- ❌ 驳回 - 红色
- ⏳ 待处理 - 黄色
