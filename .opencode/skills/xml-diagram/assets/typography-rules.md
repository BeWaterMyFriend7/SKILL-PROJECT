# 文字层级规则

所有图形必须使用稳定的文字层级。不要根据画布宽度随意缩放字号。

## 页面主标题

用于整张图的标题。

```yaml
字号: 24
字重: 加粗
对齐: 居中
颜色: 标题色
顶部边距: 20
下方间距: 8
```

示例：

```text
业务架构图设计案例
应用架构图设计案例
智能客服系统技术架构图
项目研发生命周期流程图
```

## 页面副标题

用于补充说明，可省略。

```yaml
字号: 12
颜色: 次要文字
对齐: 居中
下方间距: 24
```

## 区域标题

用于大容器标题。

```yaml
字号: 18
字重: 加粗
颜色: 区域语义色
对齐: 左侧
标题区高度: 44
```

强制规则：

```text
区域标题必须有独立安全区。
节点不得覆盖区域标题。
标题区高度不得小于 44px。
区域内节点的 y 坐标必须大于 区域 y + 标题区高度 + 内边距。
```

## 卡片标题

用于分组卡片的标题。

```yaml
字号: 14
字重: 加粗
颜色: "#ffffff" 或高对比文字色
对齐: 居中
标题高度: 34
```

## 小标签文字

用于卡片内部的小能力点、技术点、属性。

```yaml
字号: 10
颜色: 正文色
高度: 22
圆角: 6
左右内边距: 8
```

## 说明文字

用于脚注、备注、弱说明。

```yaml
字号: 11
颜色: 次要文字
对齐: 居中或左侧
```
