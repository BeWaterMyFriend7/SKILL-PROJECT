# 技术方案设计

## 执行步骤

### 第 1 步：读取需求文档

读取以下文件：
- 需求文档：`output/doc/requirements-analysis-review.md`
- 原型设计：`output/design-output/*`

### 第 2 步：技术方案设计

输出技术方案至 `output/doc/technical-design.md`

参考示例：`example/technical-design-ex.md`

内容包含：
- 技术选型及理由
- 系统架构设计
- 数据库设计
- 接口设计
- 核心流程设计
- 任务拆解及工时评估

### 第 3 步：方案评审

输出评审报告至 `output/doc/technical-design-review.md`

包含：潜在问题、优化建议

## 输出规范

- 使用 Markdown 格式
- 内容简洁明了
- 技术选型需有充分理由
