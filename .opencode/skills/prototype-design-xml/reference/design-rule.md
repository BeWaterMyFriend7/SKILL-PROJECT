1. 整体设计简约大气，避免过于花哨
2. 颜色规约
    - 主色为天蓝色 #3498db
    - 辅助色为浅灰色 #f8f9fa
    - 强调突出可以使用浅红色 #e74c3c
    - 整体配色要尽量协调
    - 无论是整体页面还是卡片突出等，要能够保持看清字体，不要乱用颜色导致字体看不清
3. 字体规约
    - 字体尽量使用系统默认字体
    - 字体大小尽量保持一致
    - 字体颜色尽量保持一致
    - 区分出不同层级的标题
4. 页面布局规约
    - 页面布局尽量保持一致
    - 页面布局尽量简洁
    - 页面仅设计web端即可，无需设计移动端，web尺寸比例为16:9
    - 核心内容要中间展示，两侧尽量留白充足，避免过分拥挤，但也不要过分留白，导致显得页面很空。
    - 页面元素间距要均匀，避免出现错落有致的情况或者大小不协调的情况。
5. XML格式规约
    - 使用 drawio XML 格式
    - 保持 mxCell id 唯一
    - 使用相对坐标布局
    - 确保连接线端点精确指向节点边缘
6. 组件规范
    - 按钮统一圆角弧度
    - 输入框统一边框样式
    - 卡片统一阴影效果
    - 列表统一间距
