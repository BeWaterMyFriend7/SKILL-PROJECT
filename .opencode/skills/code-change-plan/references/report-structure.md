# 方案报告结构规范

> 配合 SKILL.md 使用。定义最终 Markdown 方案文档必须包含的章节结构。

## FULL 模式（23 章，默认）

按以下顺序输出，不得省略。确实不适用时写"不适用：<原因>"。

1. **基本信息与仓库基线** — Plan ID、需求名称、仓库根目录、当前分支、基线 HEAD、生效的 AGENTS.md、分析范围
2. **需求分析** — 业务目标、用户可见结果、功能需求（REQ-xx）、非功能需求（NFR-xx）、非目标
3. **假设、约束与待决策项** — 假设（ASM-xx）、约束（CON-xx）、待决策项（DEC-xx，含可选方案和推荐方案）
4. **验收标准** — AC-xx 关联 REQ-xx，含场景、输入/前置条件、期望结果、验证项
5. **当前实现分析** — 相关模块和职责、关键入口/调用方/实现/外部依赖
6. **当前功能流程、调用链、数据流和配置流** — Mermaid 流程图 + 调用链 + 数据流 + 配置流 + 异常/兼容/回退路径
7. **差距分析** — GAP-xx：当前状态 vs 目标状态，差距类型（缺失能力/错误行为/重复实现/配置不一致/测试缺失/文档过时）
8. **目标方案** — 方案概述、关键设计决策（DD-xx，含被否方案）、接口与数据结构调整、配置调整、异常处理与降级、兼容和迁移策略、日志/指标/告警调整
9. **目标功能流程、调用链、数据流和配置流** — 同第 6 章结构，描述目标状态
10. **修改前后对比** — 按维度对比：用户入口、核心流程、调用链、数据流、接口/消息、配置、数据存储、脚本、测试、监控、文档、运维
11. **修改单元** — CU-xx：目标、关联需求、关联验收标准、前置依赖、涉及文件、涉及符号、影响的旧逻辑、完成条件、局部验证
12. **全项目影响矩阵** — MUST_CHANGE / SHOULD_CHANGE / OPTIONAL_CHANGE 三级，覆盖代码、测试、配置、脚本、数据、接口、文档、监控
13. **旧逻辑退役矩阵** — LR-xx：关联 CU-xx、文件路径、退役动作（删除/替换/迁移/限期兼容）、验证项
14. **文件变更清单** — 每个文件的路径、动作（修改/新增/删除/重命名）、关联修改单元、禁止事项
15. **实施顺序** — 按依赖关系的步骤顺序，每步含文件列表和完成标准
16. **测试与验证方案** — V-xx：层级（结构/静态/针对性/集成/全量/人工）、命令、期望结果
17. **需求—修改—验证追踪矩阵** — 完整追踪链 REQ → AC → CU → V
18. **发布与回滚方案** — 发布前置条件、发布顺序、灰度/监控、停止发布条件、回滚触发条件、回滚步骤、数据恢复或补偿
19. **风险** — RISK-xx：风险描述、等级（HIGH/MEDIUM/LOW）、触发条件、影响、缓解措施、验证项
20. **执行契约** — 11 条执行规则 + 停止条件 + 禁止事项
21. **完成条件** — VERIFIED / IMPLEMENTED_NOT_FULLY_VERIFIED / BLOCKED / PLAN_AMENDMENT_REQUIRED
22. **审批记录** — 审批状态、审批范围、审批人、审批时间、审批意见、方案修订版本
23. **直接执行提示词** — 审批后在新 Codex 会话中使用的完整提示词

## SIMPLE 模式（12 章）

SIMPLE 模式下，相邻章节合并输出：

| SIMPLE 章节 | 合并自（FULL） | 内容要求 |
|---|---|---|
| 1. 基本信息 | 1 | 同 FULL |
| 2. 需求分析 | 2 | 同 FULL |
| 3. 假设与约束 | 3 | 同 FULL |
| 4. 验收标准 | 4 | 同 FULL |
| 5. 当前实现与差距 | 5 + 6 + 7 | 当前实现 + 流程 + 差距，合并为一个章节 |
| 6. 目标方案与对比 | 8 + 9 + 10 | 目标方案 + 流程 + 前后对比 |
| 7. 修改单元 | 11 | 同 FULL |
| 8. 影响与清理 | 12 + 13 + 14 | 影响矩阵 + 旧逻辑退役 + 文件清单 |
| 9. 实施与验证 | 15 + 16 + 17 | 实施顺序 + 验证方案 + 追踪矩阵 |
| 10. 发布与风险 | 18 + 19 | 发布回滚 + 风险 |
| 11. 执行契约 | 20 + 21 | 执行契约 + 完成条件 |
| 12. 审批与执行 | 22 + 23 | 审批记录 + 执行提示词 |

SIMPLE 模式共 12 章，追踪关系要求不变。
