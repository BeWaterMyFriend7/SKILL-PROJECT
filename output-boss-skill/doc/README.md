# 软件台账管理平台 - 项目输出文档

## 文档说明

### PRD需求文档
- 完整的功能需求描述
- 数据模型设计
- 权限需求说明
- 可观测需求说明

### 系统架构图
包含以下内容：
- 前端展示层（Vue3）
- 后端服务层（Spring Boot）
- 数据访问层（MyBatis）
- 外部数据源（终端管理平台、应用系统管理、DevOps、服务器管理）
- 分表分库设计

### 界面原型
可交互的HTML原型，包含以下页面：
- 软件台账列表页
- 软件台账详情页
- 终端使用关系列表页
- 应用系统使用关系列表页
- 服务器使用关系列表页
- 统计大屏
- 同步监控

### 技术方案
- 主要流程设计（Mermaid图表）
- 库表设计（13张核心表）
- 接口设计
- 非功能设计保证
- 任务排期安排（约45个工作日）

## 使用说明

### 查看架构图
1. 打开 https://excalidraw.com
2. 选择 "Open" → "Open from your computer"
3. 选择 `系统架构图.excalidraw` 文件

### 查看界面原型
直接在浏览器中打开 `design-output/软件台账管理平台原型.html` 文件即可预览。

### 查看技术方案
直接阅读 `技术方案设计.md` 和 `技术方案评审报告.md` 文件。

---

## 技术栈

- **前端**：Vue3 + Element Plus
- **后端**：Spring Boot + MyBatis
- **数据库**：MySQL 8.0
- **缓存**：Redis
