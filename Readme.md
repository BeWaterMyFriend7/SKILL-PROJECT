## 项目介绍

agent skills 产品设计和技术方案设计skills示例

主要skills说明：
- requirements-analysis 需求分析
- requirements-review 需求评审
- prototype-design 原型设计，产出可交互的html
- technical-design 技术设计，产出技术方案

## 项目结构

``` markdown
.
├── .opencode
|  ├── skills  # agent skills 
├── output # skills 输出示例
```

## 项目使用

- 安装配置好opencode
- vscode 安装 opencode 插件

通过对话框即可使用相关skills
