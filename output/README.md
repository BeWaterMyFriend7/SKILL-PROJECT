# 企业软件管理系统原型

## 项目简介
这是一个企业内部软件管理系统的原型设计，旨在帮助员工查找、下载合适的软件版本，同时统一管理公司软件，避免员工私自下载不合规软件。

## 功能特性

### 页面1：软件列表页面
- 📋 软件列表展示
- 🔍 搜索功能（支持软件名称、标签、功能搜索）
- 🏷️ 多维度筛选（软件类型、功能分类、使用范围）
- 📱 响应式设计，支持移动端访问

### 页面2：软件详情页面
- 📝 详细软件信息展示
- 📦 版本管理与下载
- 📚 相关文档查看
- ⭐ 用户评价系统
- 🔄 标签页切换展示

## 文件结构
```
output/
├── index.html          # 软件列表页面
├── detail.html         # 软件详情页面
├── test.html           # 功能测试页面
├── styles.css          # 样式文件
├── script.js           # 列表页面交互脚本
├── detail-script.js    # 详情页面交互脚本
├── software-data.txt   # 软件数据存储文件
└── README.md           # 说明文档
```

## 技术实现
- **前端技术**: HTML5 + CSS3 + JavaScript (ES6+)
- **数据存储**: TXT文件作为临时数据存储
- **设计风格**: 现代化、简洁的企业级UI设计
- **响应式**: 支持桌面端和移动端

## 使用说明

### 1. 启动原型
直接在浏览器中打开 `output/index.html` 文件即可开始使用。

### 2. 功能测试（可选）
如果遇到问题，可以打开 `output/test.html` 页面进行功能测试：
- 数据加载测试
- 本地存储测试  
- 页面跳转测试

### 2. 主要操作流程
1. **浏览软件**: 在列表页面查看所有可用软件
2. **搜索筛选**: 使用搜索框和筛选条件快速找到目标软件
3. **查看详情**: 点击软件卡片进入详情页面
4. **下载软件**: 在详情页面选择合适版本进行下载
5. **查看文档**: 浏览相关文档和使用手册
6. **评价反馈**: 查看或添加用户评价

### 3. 数据管理
软件数据存储在 `software-data.txt` 文件中，采用JSON格式：
- 支持新增、修改软件信息
- 包含完整的软件属性（名称、类型、版本、文档等）
- 便于后续与后端系统集成
