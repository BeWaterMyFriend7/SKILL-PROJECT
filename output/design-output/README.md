# 软件台账管理平台 - 原型设计

## 项目介绍

本项目是软件台账管理平台的原型设计，用于展示系统的页面结构和交互逻辑。

## 包含页面

| 页面 | 文件 | 说明 |
|------|------|------|
| 首页 | index.html | 欢迎页，展示数据概览和快速入口 |
| 软件列表页 | index.html | 软件台账列表，支持多维度筛选 |
| 软件详情页 | software-detail.html | 软件详情及版本列表 |
| 终端使用关系列表 | index.html | 终端软件使用关系 |
| 应用系统使用关系列表 | index.html | 应用系统软件使用关系 |
| 服务器使用关系列表 | index.html | 服务器软件使用关系 |
| 统计大屏 | index.html | 数据统计和同步状态监控 |

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可查看原型
2. 点击顶部导航或首页快速入口切换页面
3. 点击表格行可查看详情（需配合后端）
4. 支持搜索、筛选、分页等交互

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript

## 设计规范

遵循 `design-rule.md` 中的设计规范：
- 主色：天蓝色 (#4A90E2)
- 字体：系统默认字体
- 布局：简洁大气，16:9 比例
- 配色协调，避免花哨
