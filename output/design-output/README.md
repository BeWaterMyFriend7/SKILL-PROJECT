# 软件管理平台原型设计

## 项目概述

本项目是基于需求分析评审文档设计的软件管理平台原型，采用简约大气的设计风格，实现了软件分类管理、差异化显示、权限控制等核心功能。

## 功能特性

### 1. 软件分类体系
- **一级分类**：商业软件、开源软件、自主软件
- **二级分类**：产品类、组件类
- **差异化显示**：产品类显示软件名称，组件类显示坐标

### 2. 核心页面
- **首页**：平台概览、统计信息、热门软件、快速操作
- **软件列表页**：卡片/列表视图切换、搜索筛选、分类显示
- **软件详情页**：完整软件档案、版本管理、许可证管理、用户评价

### 3. 关键功能
- **搜索筛选**：支持软件名称、坐标、功能类型搜索
- **视图切换**：卡片视图和列表视图自由切换
- **权限控制**：普通用户、部门管理员、超级管理员权限差异
- **下载管理**：产品类直接下载，组件类跳转外部下载
- **许可证管理**：到期提醒、手动自动配置结合
- **用户评价**：发布即显示，实时交互
