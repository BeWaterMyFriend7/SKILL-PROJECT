软件简介

Spring Boot 是由 Pivotal 团队提供的全新框架，其设计目的是用来简化新 Spring 应用的初始搭建以及开发过程。该框架使用了特定的方式来进行配置，从而使开发人员不再需要定义样板化的配置。

功能特性

快速构建 快速创建独立的Spring应用
内嵌服务器 内嵌Tomcat、Jetty或Undertow
自动配置 自动配置Spring功能
生产就绪 提供健康检查、指标审计等功能

适用场景

适用于构建微服务架构应用、RESTful API服务、响应式应用等场景。特别适合需要快速启动和简化配置的企业级应用开发。

Maven依赖配置

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
    <version>3.2.1</version>
</dependency>

application.yml配置示例

server:
  port: 8080
  servlet:
    context-path: /app

spring:
  application:
    name: myapp
    
logging:
  level:
    org.springframework.web: INFO
    com.example: DEBUG

启动类示例

import org.springframework.boot.SpringApplication;
import org.springframework.boot.autoconfigure.SpringBootApplication;

@SpringBootApplication
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}

常见问题

Q: 如何打包为JAR？ 使用 mvn clean package 命令
Q: 如何修改端口？ 在application.yml中设置server.port
Q: 如何关闭自动配置？ 使用@SpringBootApplication(exclude={...})
